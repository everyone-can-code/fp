Functor

So, What are Functors? Functors are the containers or type that can be used with ‘map’ function.

Example:

Container.prototype.map = function (f) {
  return Container.of(f(this.$value));
};

Container.of(2).map(two => two + 2);

[1, 2, 3].map((num) => num + 2);
