Setoid

A value which has a Setoid must provide an equals method. The equals method takes one argument:

Graphic

graph LR; A[Setoid]

Example:

Ramda:

import { equals } from 'ramda'

equals([1, 2], [1, 2]) // true
equals([1, 2], [0]) // false

Ramda (currying):

import { equals } from 'ramda'

equals([1, 2])([1, 2]) // true
equals([1, 2])([0]) // false
