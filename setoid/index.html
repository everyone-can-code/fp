Setoid

A value which has a Setoid must provide an equals method. The equals method takes one argument:

Make array to Setoid:

Array.prototype.equals = (arr) => {
  const len = this.length

  if (len !== arr.length) {
    return false
  }

  for (let i = 0; i < len; i++) {
    if (this[i] !== arr[i]) {
      return false
    }
  }

  return true
}

;[1, 2].equals([1, 2]) // true
;[1, 2].equals([0]) // false
